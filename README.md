# 🎥 Real-Time Video Communication Platform

A real-time video communication platform inspired by Omegle, built using Node.js, Express.js, EJS, Socket.io, and WebRTC.

The application enables users to instantly connect with random users for live video conversations, text chat, screen sharing, and interactive communication in real time.

## ✨ Features

### 📹 Real-Time Video Calling
- Peer-to-peer video communication using WebRTC
- Real-time audio and video streaming
- Instant random user matching

### 💬 Live Chat
- Real-time messaging using Socket.io
- Emoji support for interactive conversations
- Smooth instant communication experience

### 🖥️ Screen Sharing
- Share screen during live video sessions
- Dynamic media stream switching
- Real-time screen stream handling

### 🔄 User Matching System
- “Next User” functionality for random matching
- Instant disconnect and reconnect workflow
- Real-time room and connection management

### ⚡ Performance & UX
- Responsive and user-friendly interface
- Optimized socket event handling
- Smooth media stream experience

## 🔗 WebRTC Communication Flow
- User joins the platform
- Socket.io handles user matchmaking
- WebRTC peer connection is established
- Media streams are exchanged between peers
- Real-time communication begins
- Users can switch to next connection instantly
- 
## 🖥️ Screen Sharing Flow
- User selects screen sharing option
- Browser media devices API captures screen stream
- Stream is shared through WebRTC peer connection
- Remote user receives updated live stream instantly
